# Discord Bot - 24/7 Hosting

This bot includes:
- Blackjack game
- Wordle game
- Mines game (minesweeper with betting)
- User info commands
- Balance system
- **Tower Game** Coming Soon

- /ping
- /userinfo
- /balance
- /blackjack
- /wordle
- /mines
- /clearmines

## Setup
1. Set DISCORD_TOKEN environment variable
2. Set GUILD_ID environment variable (optional)
3. Run: python main.py
